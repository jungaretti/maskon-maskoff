# Mask On, Mask Off

Mask On, Mask Off analyzes images to identify faces and masks.

## Installation

Create a Python virtual environment, switch to it, and install dependencies.

```bash
python3 -m venv venv
. ./venv/bin/activate
pip3 install -r requirements.txt
```
